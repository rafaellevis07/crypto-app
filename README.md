# Crypto

Final Project using API from Coingecko and Firebase database and authentication

### Tech Stack:

- HTML
- Tailwind
- JSX
- React

### Third Party Libraries & APIs:

- react-router-dom
- react-sparklines
- dompurify
- Coingecko API

### Install and Run:

- Run 'npm start' on the crypto-app folder folder
